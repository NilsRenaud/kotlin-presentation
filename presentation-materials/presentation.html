What's Kotlin ?

Basics

  • Functional & Object Oriented Programmation language
  • Strong typing
  • Created in 2011, 1st stable release in 2016
  • By Jetbrain (Intellij)
  • Works on top of the JVM
  • ... but also the Javascript VM
  • Inspired from Java, Scala, Groovy, C#...
  • Google I/O 2017 : officially announced a first class support on Android
the others First class citizen on Android are C++ and Java.

History

  • They really liked Scala...
  • ... but it had way tooooo long compile time !
  • So they decided to build a new language which had :
    • an Industrial-strength OO language
    • "better language" than Java
    • fully interoperable with Java Code
  • Kotlin is an island near St-Petersbourg
Fully operable allow a step by step migration !

How to use it ?

  • Have a JDK6+ installed / JS Interpreter
  • Write your code in Intellij/Eclipse/Android Studio/Emacs
  • Compile it with Gradle/Maven/Ant using Kotlin plugins
  • That's it !

Adoption

  • Backend : AWS, Pinterest, Coursera, Netflix, Uber, Trello...
  • Android : Expedia, Flipboard, Pinterest, Square...

Why Kotlin ?

"Better than java" they said...

  • NO MORE ";"
  • Safe
  • Consice
  • Extension Functions
  • lambda included (Android is stuck on Java7)
  • Fantastic standard library
  • Mature
  • ... Fun...
Safe : No more NPE, Enforced Immutability Concise : Sata classe as One liner, Type Inference, String Template

Functional Programming

  • Avoid mutable data / Changing state
  • Say what you want to do, not how
  • The result of a function depends only of its args
